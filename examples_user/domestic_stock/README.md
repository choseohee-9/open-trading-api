# 한국투자증권 Open API 기반 자동매매 시스템

이동평균선 교차(골든크로스 / 데드크로스) 전략을 사용해 국내주식을 자동으로
매매하는 시스템입니다. 한국투자증권(KIS) Open API의 **모의투자** 환경에서 동작합니다.

## 전략 개요

이 시스템은 가장 대표적인 추세추종 전략인 **이동평균선 교차 전략**을 사용합니다.

- **단기 이동평균선(5일)** 이 **장기 이동평균선(20일)** 을 아래에서 위로 돌파하면
  상승 추세 전환으로 보고 **매수(골든크로스)** 합니다.
- 단기선이 장기선을 위에서 아래로 돌파하면 하락 추세 전환으로 보고
  **매도(데드크로스)** 합니다.

```
       골든크로스(매수)              데드크로스(매도)
            ↑                            ↓
   단기선 ──╱── 장기선          장기선 ──╲── 단기선
```

## 대상 종목

| 종목코드 | 종목명 |
|----------|--------|
| 005930   | 삼성전자 |
| 000660   | SK하이닉스 |
| 035720   | 카카오 |

종목은 `auto_trader.py` 의 `TARGET_STOCKS` 딕셔너리를 수정해 자유롭게 바꿀 수 있습니다.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `auto_trader.py` | 실시간 자동매매 메인 프로그램. 장중 60초마다 신호를 확인하고 모의 주문 실행 |
| `backtest.py` | 과거 데이터로 전략을 검증하고 거래 기록(`backtest_log.csv`)을 생성 |
| `trade_log.csv` | 실시간 매매 거래 기록 (자동 생성) |
| `backtest_log.csv` | 백테스트 거래 기록 (자동 생성) |

## 동작 방식

1. 한국투자증권 Open API로 인증(모의투자)합니다.
2. 대상 종목의 최근 일별 종가를 조회해 단기·장기 이동평균을 계산합니다.
3. 교차 신호(BUY / SELL / HOLD)를 판별합니다.
4. 신호가 발생하면 시장가로 모의 주문을 실행합니다.
5. 모든 거래 내역을 CSV 파일에 시각, 종목, 신호, 가격, 수량과 함께 기록합니다.
6. 중복 주문을 막기 위해 직전 신호와 동일하면 주문하지 않습니다.

## 실행 방법

### 사전 준비
- Python 3.11 이상
- 한국투자증권 Open API **모의투자** 앱키 / 앱시크릿
- `~/KIS/config/kis_devlp.yaml` 설정 완료
- 이 파일들을 KIS 샘플 저장소의
  `examples_user/domestic_stock/` 폴더 안에 둡니다
  (공통 모듈 `kis_auth.py`, `domestic_stock_functions.py` 를 import 하기 때문)

### 백테스트 (언제든 실행 가능)
과거 데이터로 거래 기록을 만듭니다. 장이 닫혀 있어도 동작합니다.

```bash
uv run python backtest.py
```

실행하면 `backtest_log.csv` 에 매수/매도 시점이 기록됩니다.

### 실시간 자동매매 (장중 실행)
장 운영 시간(평일 09:00~15:30)에 실행하면 60초마다 신호를 확인합니다.

```bash
uv run python auto_trader.py
```

종료하려면 `Ctrl + C` 를 누릅니다.

## 거래 기록 예시

`backtest_log.csv` / `trade_log.csv` 는 다음과 같은 형식입니다.

```
날짜,종목코드,종목명,신호,가격,수량
20250512,005930,삼성전자,BUY,78500,1
20250603,005930,삼성전자,SELL,81200,1
```

## 주의사항

- 본 시스템은 **모의투자 환경** 전용으로 작성되었습니다. 실전 계좌로 사용 시
  실제 자산이 거래되므로 주의가 필요합니다.
- 이동평균 교차 전략은 학습·과제 목적의 단순 전략이며, 실제 수익을 보장하지 않습니다.
- 모의투자 계좌는 API 호출 제한이 있어 종목 수나 주기를 과도하게 늘리면
  호출 초과 오류(`EGW00201`)가 발생할 수 있습니다.
